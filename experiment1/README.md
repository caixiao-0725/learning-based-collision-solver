#实验1记录

目标：给定两个固定位置的已经相交的三角形，尝试在一定的步长内把它俩分开

两种想法：1.给每个action一个最大的移动距离，这个可以用softmax实现
         2.在reward函数里面加入action的模长，来限制action的大小

在使用方法2的时候，前面几个训练epoch会产生非常小的reward，-1e9的数量级都有，输出action以后发现，action非常大，尝试多训练一会儿看看能不能work。
